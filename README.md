# RC_Light

## Pin Connect

- Receiver
	- CH1 (Thottle) -> Pin3
	- CH2 (Steering) -> Pin 2

- LED
	- Head -> Pin 10
	- Back -> Pin 11
	- Break -> Pin 9
	- Turn Left -> Pin 7
	- Turn Right -> Pin 8
